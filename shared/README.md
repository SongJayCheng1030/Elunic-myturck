# Shopfloor.io shared library
===

This folder contains various shared libraries for the shopfloor.io application. 

## 📡 Shared library for Angular

The folder `./shared/angular/projects/shopfloorio-common/` contains the angular shared library with the following common elements to use:

### 🧭 Components

 - `<lib-header appName="Name of the current app"></lib-header>` -- A general header component for the app
 - `<lib-loader></lib-loader>` -- A loading indicator component
